# Varlog 

Varlog는 일관성을 제공하는 분산 로그 저장소이다. Varlog를 활용해서 고성능으로
트랜잭션 이벤트 처리를 하고 더 복잡한 분산 시스템을 간단하게 구축할 수 있다.
